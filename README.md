# messcript

`messcript` is an installable command-line mess detector for JavaScript and
TypeScript source.

```sh
npm install --save-dev messcript
npx messcript --help
```

The executable accepts the sibling-tool command shape:

```text
messcript <paths> <format> <ruleset[,ruleset...]> [options]
```

Run `messcript --help` for the available options.
